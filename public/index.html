FCC API: Image Search Abstraction Layer

User Stories:
  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.
    • Example usage:
  • To search for 'game of thrones' images try https://image-search-info.herokuapp.com/search/game of thrones
  • You can get 5 results by adding ?offset=5 like https://image-search-info.herokuapp.com/search/game of thrones?offset=5
  • To see the last 10 searches go to https://image-search-info.herokuapp.com/latest
    • Example output:
  • Output for the image results [ { url: 'http://...', snippet: '...', thumbnail: 'http://...', context: 'http://...' }, ... ]
  • Output for the most recent searches [ { term: '...', when: '...' }, ... ]
